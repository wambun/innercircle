# Insurance Template - Page Layouts Reference

## Homepage Layout

### 1. Navigation Bar
- **Position**: Fixed top
- **Content**: Logo | Nav Links | CTA Button
- **Background**: White/transparent
- **Height**: ~80px

### 2. Hero Section
- **Padding**: 200px top, 100px bottom
- **Content**:
  - Badge/label (mint background)
  - Main headline (large, bold)
  - Subheadline paragraph
  - Primary CTA button
  - Secondary link/button (optional)
  - Hero image/illustration

### 3. Services Overview
- **Padding**: 130px vertical
- **Layout**: 3-column grid
- **Content per card**:
  - Icon
  - Service title
  - Short description
  - Learn more link
- **Card style**: White bg, shadow, rounded corners

### 4. About/Mission Section
- **Padding**: 130px vertical
- **Layout**: 2-column (text | image)
- **Content**:
  - Section badge
  - Headline
  - Mission statement
  - Key points/features list
  - CTA button

### 5. Features/Benefits Section
- **Background**: Soft gray (#fafafa)
- **Padding**: 130px vertical
- **Layout**: Grid or list with icons
- **Content**:
  - Feature icon
  - Feature title
  - Feature description

### 6. Testimonials Section
- **Padding**: 130px vertical
- **Layout**: 3-column cards
- **Card content**:
  - Quote text
  - Client name
  - Client title/company
  - Optional: star rating

### 7. Stats/Numbers Section
- **Background**: Teal or dark
- **Layout**: 4-column on desktop
- **Content**:
  - Large number
  - Label/description

### 8. CTA Section
- **Background**: Dark navy (#003127)
- **Padding**: 100px vertical
- **Content**:
  - Headline
  - Subtext
  - CTA button (teal)
  - Optional: form

### 9. Footer
- **Background**: Dark navy (#003127)
- **Layout**: Multi-column
- **Content**:
  - Logo
  - Navigation links
  - Contact info
  - Social links
  - Copyright
